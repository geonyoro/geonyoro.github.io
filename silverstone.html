Silverstone Tyre Tool

This is a tool created to enable customers to search for the right tyres that fit their vehicle. I created it as part of my work under Luvaria Systems. The tool can be found here.

It allows:
  • Searching based on Vehicle Make
  • Searching based on Model
  • Searching based on Engine
  • Administrator Upload/Download of Database File

Implementation

It took about 19 hours of development to get it running. It implements CodeIgniter in the background.

The administrator uploads files Excel/CSV File containing a list of tyres and another containing a list of vehicles, engines, details etc. During the upload process the documents are parsed and fed into the database.

For the front-end, JQuery is used to to autocomplete the fields as one searches for the appropriate details and filters them by that.

Features

The Homepage with the necessary fields. Inspired by and developed using Grid Framework Bootstrap.
Results of matching tyres are presented to the user!
Admin Interface by which (s)he is capable of uploading the various files, adding users, changing passwords etc.
