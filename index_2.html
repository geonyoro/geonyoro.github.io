Back and Low

With a background in Mechatronic Engineering, coding low and coding hard-core is George's strength. Beginning in Python, graduating to C, and picking up a host of numerous other languages along the way including PHP, Java, x86 Assembly, C++, languages for the full web stack. He has experience coding in C for Arduino, Linux as well as PIC microcontrollers as well as using Python on the Raspberry Pi.

Front and Centre

Having started Desktop GUI programming for personal fun with Python making cool toys for himself, among the earliest jobs George had an opportunity to do was with Tkinter, the default toolkit that comes with Python. He later graduated to the likes of GTK that handles all the running of Linux front-end. As for the web, over 2 years experience with web development for personal and commercial use has seen George grow his skill in front-end development to impressive levels, including making and overhauling the site http://launchform.com as well as developing this particular site, besides working with other clients.
