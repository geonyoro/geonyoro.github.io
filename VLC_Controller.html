VLC Wifi Controller

This is an app I created for my Ubuntu Linux to thoroughly enhance my laziness. One can setup a hotspot on a phone or simply connect to the same wifi network a computer is connected to, and navigate using the browser to a certain ULR presented.

From there, you have a web based remote controller. And it is not limited by range or even clear line of sight between the remote and the screen.

It allows:

  • Display of current playing file.
  • Forward/Pause/Play/Rewind operations
  • Volume Manipulations
  • Switch off the app and VLC!

Implementation

It took about 5 hours of development to get it running. It leverages the power of dBus interprocess communication API for communicating with VLC in the background.

The server process that runs the web interface is based on Flask. The app is written in python and thus uses the dbus python API. It can only run on Linux since Dbus is only based on linux.
